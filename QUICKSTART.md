# ⚡ 快速开始指南

## 🚀 5分钟启动项目

### 步骤1：安装依赖（首次运行）

```bash
cd e:\bookking_kit
npm install
```

等待依赖安装完成（约1-2分钟）

### 步骤2：启动开发服务器

```bash
npm run dev
```

浏览器自动打开 http://localhost:3000

### 步骤3：开始使用

1. **添加玩家**
   - 点击底部"用户"标签
   - 点击"添加用户"
   - 输入昵称（例如：张三、李四、王五、赵六）
   - 添加4个玩家

2. **创建牌局**
   - 返回首页
   - 点击"新建牌局"
   - 选择游戏类型（麻将/扑克/桥牌）
   - 选择参与玩家
   - 点击"创建牌局"

3. **记录得分**
   - 自动进入牌局详情页
   - 点击"记录本局得分"
   - 为每位玩家输入得分（赢为正数，输为负数）
   - 确保得分总和为0
   - 点击"确认提交"

4. **查看排名**
   - 积分榜实时更新
   - 自动按得分高低排序
   - 显示每局得分变化

5. **结束牌局**
   - 点击底部"结束牌局"
   - 查看最终排名
   - 点击"截图保存"分享结果

## 💡 使用技巧

### 快速输入得分

记录得分时，点击"±100"按钮快速调整：
```
初始值: 0
点击[+100] → 100
点击[+100] → 200
点击[-100] → 100
```

### 得分规则

所有玩家得分总和必须为0：
```
✅ 正确示例：
张三: +100
李四: +50
王五: -50
赵六: -100
总和: 0 ✓

❌ 错误示例：
张三: +100
李四: +50
王五: -50
赵六: -50
总和: +50 ✗（无法提交）
```

### 多人游戏

支持2-8人游戏：
- 麻将：通常4人
- 斗地主：3人
- 扑克：2-8人

## 🔧 常见问题

### Q1: 数据会丢失吗？

**A**: 不会。使用IndexedDB存储在浏览器本地，除非清除浏览器数据，否则永久保存。

### Q2: 可以同时进行多个牌局吗？

**A**: 可以。每个牌局独立管理，可以同时创建多个牌局。

### Q3: 如何修改已记录的得分？

**A**: 当前版本不支持修改历史记录。建议记录前仔细核对。

### Q4: 可以删除牌局吗？

**A**: 可以。在历史记录页面长按牌局卡片即可删除。

### Q5: 如何分享给朋友？

**A**:
1. 部署到GitHub Pages后，分享网址
2. 或在牌局结束后截图分享结果
